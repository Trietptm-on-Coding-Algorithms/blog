我在这里列举一些在我看来非常棒的开源项目和书籍

Win32

hasherezade/libpeconv

hasherezade/pe-sieve

fireeye/flare-vm

fireeye出品, 用于恶意软件分析, 应急响应和渗透测试的高可定制化基于Window的发行版. 只需要运行powershell脚本即可配置环境.

cetfor/AntiDBG

代码清晰且自包含, 里面对一些反调试技术的原理和绕过方法都有言简意赅的说明. 适合学习

ARM

VisUAL

一个跨平台且高度可视化的ARM模拟器, 用于更好地学习ARM汇编语言.

C++

CCareaga/heap_allocator

作者@CCareaga为OS爱好者而编写的一个简单堆分配器. 代码200行左右, 力图易于理解, 适合那些OS开发的初学者以及那些对malloc和free的简单函数实现感兴趣的人学习.

pbohun/stack-vm-tutorials

实现一个简单的基于栈的虚拟机教程. 非常简单易懂并且作者在Youtube上配有教学视频.

Exploit

nnamon/linux-exploitation-course

一个学习linux漏洞利用的课程.

Machine Learning

jindongwang/transferlearning-tutorial

迁移学习简明教程

IDA

zyantific/IDASkins

cm108070/ida_dark_colors

IDAPython Book
